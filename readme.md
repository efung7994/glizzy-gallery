# Background
Glizzy Gallery is a website to create your very own glizzy! What is a glizzy? It's another word for a hot dog! This site uses Python and Django along with other technologies to create a simple CRUD app.

  

# Technologies Used

- Django
- PostgreSQL
- HTML
- CSS
- Git
- Dockerfile
